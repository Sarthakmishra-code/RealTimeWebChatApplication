# Real-Time Chat Application

A simple real-time chat application built using the MERN stack.

Users can register, log in securely using JWT authentication, and send messages instantly using Socket.IO. Messages are stored in MongoDB, and the frontend is built with React and Tailwind CSS with a clean and minimal interface.

## Tech Stack
- React
- Tailwind CSS
- Node.js
- Express
- MongoDB
- Socket.IO
- JWT Authentication

## Features
- User registration and login
- Real-time messaging
- Message storage in MongoDB
- Minimal and responsive UI

## Setup
1. Clone the repository
2. Install dependencies for both client and server
3. Add environment variables
4. Run the backend and frontend servers
